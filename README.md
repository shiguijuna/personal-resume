# shiguijun 个人博客 / 作品集

基于 Nuxt 3 + Vue 3 + TypeScript + Sass 构建的个人博客与作品集。

## 功能

- 个人简历与作品展示
- 技术栈展示（Vue / Nuxt / 小程序 / 中后台）
- 项目经历时间线
- 博客文章预览
- 自定义主题色

## 快速开始

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
pnpm preview
```

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Nuxt 3, Vue 3 |
| 语言 | TypeScript |
| 样式 | Sass, CSS Variables |
| 组件库 | Vant 4 (Nuxt) |
| 工具 | ESLint, Prettier |
